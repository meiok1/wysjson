# wysJSON 开发讨论日志

## 项目概述
**wysJSON** 是一个 VS Code 插件，允许用户在代码编辑器中选中 JSON，右键点击 "wysJSON" 以类 Excel 表格方式编辑 JSON 数据。

## 技术方案

### 核心设计
- **文件结构**:
  - `index.html` — 主入口页面（完整功能）


### 表格渲染逻辑（Recursive Nested Table）
1. 解析 JSON → 遍历数据：
   - 如果是 **数组** → 渲染多行表，每元素一行
   - 如果是 **对象** → 渲染单行表，键为列名
   - 如果是 **基本类型** → 可编辑单元格
   - 如果是 **嵌套对象/数组** → 单元格内递归渲染子表格
2. 列标题 = 字段名（JSON key）
3. 单元格值 = 字段值（JSON value）
4. 字段顺序 = 列顺序（按 Object.keys 顺序）

### VS Code 插件架构（计划）
```
Extension 端           Webview 端
┌─────────────┐       ┌──────────────────┐
│ 选中 JSON    │──→   │ 接收 JSON 数据    │
│ 右键菜单     │       │ 渲染嵌套表格      │
│ 打开 Webview │       │ 用户编辑          │
│             │←──   │ 发送更新后 JSON   │
│ 写回编辑器   │       └──────────────────┘
└─────────────┘
```
