About {}jsonOK

English

{}jsonOK is a static JSON table editor for nested objects, arrays, type switching, export, and focused cell editing.

This page provides a public introduction for the site and helps with search engine indexing.

中文

{}jsonOK 是一个静态 JSON 表格编辑工具，支持嵌套对象、数组、类型切换、导出和局部编辑。

本页面用于站点介绍和搜索引擎收录，部署后可作为公开首页的一部分。
